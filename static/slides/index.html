Drupal.Sapper.Svelte.GraphQL

Synopsis

  • Drupal 8 + GraphQL
  • Sapper + Svelte

Steps

  • Add GraphQL to Drupal
  • Enable CORS
  • Set permissions
  • Create Sapper app
  • Create /article and /article/* routes
  • Make GraphQL queries in the preload methods

Benefits

  • Performance.
  • Better UX (prefetch, transitions, etc.)
  • CMS for content, JavaScript for frontend.
  • Easier to customize.
  • More secure? Smaller attack surface.
